# 爪爪运 · 宠物玄学小程序 — 产品说明文档 v1.0

---

## 一、产品概述

**产品名称**：爪爪运（备选：毛毛签）  
**产品类型**：微信小程序  
**核心定位**：宠物玄学算命小程序，用户上传宠物照片或填写特征信息，由 AI 生成有趣、符合"玄学"风格的运势解读  
**目标用户**：18-35岁宠物主人，喜爱萌宠内容、娱乐化玄学、分享社交  
**核心情绪价值**：趣味、温暖、惊喜、共鸣、分享欲

**当前版本（v1.0）实现功能**：宠物算命（主线功能）  
**规划功能（v2.0+）**：星座运势、主人与宠物匹配度、宠物姻缘

### 技术栈
- **框架**：微信原生小程序（WXML + WXSS + JS）
- **后端**：微信云开发（CloudBase）— 无需独立服务器
- **AI 接口**：阿里云百炼 `qwen-vl-plus`（图片+文字全支持，单模型方案）
- **图片处理**：`wx.chooseMedia` + `wx.compressImage` 压缩至 75% 质量后转 base64
- **状态管理**：页面级 data + globalData（app.js）
- **数据存储**：云数据库（用户次数、历史记录）+ wx.storage（本地缓存）

### 云数据库集合
```
user_usage       # 每日使用次数记录
fortune_records  # 算命历史（每用户最多20条）
```

### 4.3 上传照片页 `pages/upload-photo`

**布局结构**：
1. 动态标题（如"狗狗鼻纹算命"）+ 今日剩余次数徽章
2. 照片上传区（虚线框，上传后显示预览 + 删除/重选按钮）
3. 拍摄小贴士（正确/错误示范两列对比）
4. 宠物信息表单：
   - 宠物名字（**必填**）
   - 性别（**必填**：公/母）
   - 品种（⭐强烈推荐，影响毒舌精准度）
   - 年龄（⭐强烈推荐，影响命格解读深度）
5. 底部"开始算命 🔮"按钮

**校验逻辑**：
- 无照片 → Toast 提示
- 无名字或性别 → Toast 提示
- 无品种 → 弹窗确认"是否继续"（不强制）
- 次数用完 → 页面加载时前置检查，禁用提交按钮

**图片处理**：
```js
wx.chooseMedia({ sizeType: ['compressed'] })
  → wx.compressImage({ quality: 75 })
  → readFileSync(path, 'base64')
```

## 五、云函数设计 `cloudfunctions/aiProxy`

### 统一入口

```js
exports.main = async (event) => {
  const { OPENID } = cloud.getWXContext();  // 自动获取，无需登录
  switch (event.action) {
    case 'fortune':    return handleFortune(OPENID, event);
    case 'checkUsage': return handleCheckUsage(OPENID);
    case 'getRecords': return handleGetRecords(OPENID);
  }
};
```

### AI 模型方案
- **模型**：阿里云百炼 `qwen-vl-plus`（支持图片+文字，单模型全搞定）
- **费用**：~¥0.002/次（含图片）；~¥0.001/次（纯文字）
- **新用户免费额度**：100 万 Token，约可支撑 600+ 次算命
- **迁移路径**：未来换模型只需改 `baseURL` / `apiKey` / `model` 三行

### API Key 配置
在微信云开发控制台 → 云函数 → aiProxy → 环境变量 中设置：
```
QWEN_API_KEY = your_key_here
```
**绝不写进代码，绝不提交到 Git。**

## 六、喵仙道人人设

**喵仙道人**是一只修炼千年、洞悉宇宙命理的老猫，以算命为业、以毒舌为乐。

| 宠物 | 态度 | 命格名称风格 |
|------|------|------------|
| 狗狗 | 傲慢轻视但认真算，毒舌后有正向转折 | 摇尾乞怜型、过于热情型 |
| 猫咪 | 同族欣赏，标准极高，挑剔式赞美 | 月华公主命、御风浪子型 |
| 兔子 | 温和欣赏，带"别太敏感"的劝慰 | 月兔使者型、清风听者命 |
| 小宠 | 见怪不怪，因物种随机切换态度 | 宇宙微粒型、花生魂命格 |

所有 System Prompt 详见 `cloudfunctions/aiProxy/utils/prompts.js`。

## 七、今日灵签

60 条预置签文，按日期取模循环：
```js
function getTodaySign() {
  const today = new Date();
  const dayOfYear = Math.floor((today - new Date(today.getFullYear(), 0, 0)) / 86400000);
  return FORTUNE_SIGNS[dayOfYear % FORTUNE_SIGNS.length];
}
```
完整签文列表见 `miniprogram/utils/fortune.js`。

## 十、错误处理规范

| 场景 | 处理方式 |
|------|----------|
| 未上传照片提交 | Toast"请先上传宠物照片 📸" |
| 网络/云函数失败 | 弹窗"喵仙道人打盹了，再试一次？"+ 重试按钮 |
| AI 返回格式异常 | 自动 Fallback 到预置默认结果 |
| 今日次数用完 | 页面加载时前置检查，禁用按钮 + Modal 提示 |
| 相机/相册无权限 | Modal 引导"去设置"开启权限 |

## 十三、开发注意事项

1. **API Key 安全**：只存云函数环境变量，绝不进代码和 Git
2. **图片压缩**：上传前压缩至 75% 质量，节省 Token 费用
3. **内容安全**：AI 生成内容建议经过 `wx.msgSecCheck` 检测（上线前接入）
4. **iPhoneX 适配**：底部按钮区加 `padding-bottom: env(safe-area-inset-bottom)`
5. **深色模式**：暂不支持，`app.json` 中设置 `"darkmode": false`
6. **云数据库权限**：`user_usage` 和 `fortune_records` 设为"仅创建者可读写"
